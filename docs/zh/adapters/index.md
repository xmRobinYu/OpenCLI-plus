# 所有适配器

运行 `opencli list` 查看完整命令列表。

## 本次新增内容抓取能力

| 站点 | 命令 | 说明 |
|------|------|------|
| 飞书 / Lark | `opencli feishu doc --url <文档页>` | 导出单个飞书文档 / Wiki 页面为 Markdown |
| 知识星球 | `opencli zsxq download <topic_id\|topic_url>` | 导出单条话题为 Markdown，包含评论和图片 |
